# 职场心理学研究 — Workplace Psychology

> AI Think Tank 驱动的职场心理学研究项目
> 两个 AI 角色：心理咨询专家 + 职场社交 Coach

## 研究方向

### 核心问题
在中国公司的职场环境中：
- 跟领导相处的心理学是什么？
- 跟同事相处的人际关系心理是什么？
- 业界经典理论和著作有哪些？
- 如何运用这些理论提升职场关系？

### 两个 AI 角色
1. **心理咨询专家** — 分析深层心理动机、情绪管理、认知模式
2. **职场社交 Coach** — 实战策略、话术、场景应对方案
